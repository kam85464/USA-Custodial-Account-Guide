# UGMA vs UTMA

| Feature | UGMA | UTMA |
|--------|------|------|
| Allowed assets | Cash, stocks, ETFs, mutual funds | All UGMA assets + property, royalties |
| Complexity | Simple | More flexible |
| Best for | Financial assets | Mixed assets |

## Recommendation
UGMA is sufficient for most families investing in ETFs, stocks, and mutual funds.
